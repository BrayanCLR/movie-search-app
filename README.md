# 🎬 Buscador de Películas

Este es un proyecto web que permite buscar películas usando la API de TMDB.

## 🚀 Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- TMDB API

## 🔎 Funcionalidades

- Buscar películas por nombre
- Mostrar resultados dinámicos
- Interfaz simple y responsive

## 📦 API utilizada

Este proyecto utiliza la API de The Movie Database (TMDB):
https://www.themoviedb.org/

## 🎯 Objetivo del proyecto

Practicar JavaScript, consumo de APIs y manipulación del DOM.
